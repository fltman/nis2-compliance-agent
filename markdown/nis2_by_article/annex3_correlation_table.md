---

## Key Changes: NIS1 → NIS2

| Aspect | NIS1 | NIS2 |
|--------|------|------|
| **Scope** | OES + DSP (limited) | Essential + Important (broader) |
| **Entity identification** | Member State designates | Self-identification (size-cap) |
| **Supervision** | One approach | Differentiated (essential vs important) |
| **Incident reporting** | 72h notification | 24h early warning + 72h notification |
| **Fines** | Not specified | €10M/2% (essential), €7M/1.4% (important) |
| **Management liability** | Not explicit | Explicit (Article 20) |
| **Supply chain** | Not explicit | Explicit requirement (Article 21(2)(d)) |
